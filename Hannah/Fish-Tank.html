Our Project Fish Tank

When you first see "Fish Tank Webpage",
 you'd probably think of the normal "picture" fish tank,
but this is waaaay different. Our Project Fish Tank is special.
Keep reading this and the updates to find out why...

Starting the object oriented fishes:

First we practiced functions,
then we made them object oriented.
We copy-pasted the basic fish code from the challenge
into our fish prototype. We called the object
oriented function in a few of loops,
with randoms, and we did it inside a draw = function,
and it made our specified number of fish
in random spots on the screen, with random colors,
with random-ish sizes! Go see!

Making a fish ourselves:

Making a fish ourself means, no more "basic fish code"
copy- pasting. That fish code is much more confusing than
the name implies. It is lots of turning hard coded numbers into variables,
and positioning/resizing with variables.
It is cool. Now, please, come see!

Bezier practice:

After we did the object oriented fishes,
we realized that we needed seaweed in our fish tank!
The only way to make the seaweed look the way we wanted,
was to use beziers... Beziers were much more cornfusing than we thought.
We had to do: bezier(x1, y1, x2, y2, x3, y3, x4, y4); !!!
So we decided to practice the beziers first.
five beziers a day, and they had to look like seaweed,
but with different colors and places.
My first cool twist on beziers
My second bezier art program!

Seaweed and Fish Tail Flapping:

Our fish tank finally got seaweed,
by us doing what we did for the fish, but for seaweed,
and it was all on the bottom. They were all different lengths.
It looked really cool, but we thought it was missing movement.
These weren't fish statues!
So, we made the tails flap at different rates with
some weird PI thing that I still don't completely understand.
All I know is that it was cool!
Go see my amazing tail flapping fish tank!
